# Apresentacao Eletromecanica - Fonte Regulada 5V

Landing page criada para apresentar o projeto de uma fonte de alimentacao regulada de 5V, desenvolvido para a materia de Eletromecanica.

## Estrutura do projeto

```text
.
|-- index.html
|-- css/
|   `-- style.css
|-- js/
|   `-- script.js
`-- assets/
    `-- images/
        `-- pessoais/
```

## Como abrir

Abra o arquivo `index.html` diretamente no navegador.

## Onde colocar imagens pessoais

Coloque suas imagens nesta pasta:

```text
assets/images/pessoais/
```

Exemplo de caminho para usar no HTML:

```html
<img src="assets/images/pessoais/minha-foto.jpg" alt="Descricao da imagem">
```

Use nomes simples, sem espacos e sem acentos, por exemplo:

```text
aluno.jpg
montagem-circuito.jpg
fonte-regulada.jpg
```

## Arquivos principais

- `index.html`: estrutura da landing page.
- `css/style.css`: estilos visuais, paleta de cores, responsividade e animacoes.
- `js/script.js`: navegacao entre secoes, progresso, menu mobile e animacoes de entrada.

## Observacao

A pasta `assets/images/pessoais/` inclui um arquivo `.gitkeep` apenas para garantir que o Git mantenha a pasta mesmo quando ela estiver vazia.
